Crucible12

A fully local, fully offline AI coding agent — tuned specifically for an RTX 5090, a Ryzen 9 9950X3D, and 64GB of DDR5-6400. No API keys, no cloud, no usage limits.

View on GitHub Quick Start

Built on what already works

Crucible12 doesn't reinvent the agent loop — it pairs OpenCode, the leading open-source terminal coding agent, with a hardware-tuned llama.cpp inference backend and a model chosen specifically to exploit this exact GPU + CPU + RAM combination.

Why this hardware, specifically

RTX 5090
32GB GDDR7 · Blackwell
Attention layers, shared experts, most routed experts
Ryzen 9 9950X3D
16C / 32T · 3D V-Cache
Computes MoE experts that don't fit on the GPU
64GB DDR5-6400
CL32, dual-channel
Capacity for overflow experts the GPU alone can't hold

Quick start

git clone https://github.com/Mr-Pythoneer/Crucible12.git
cd Crucible12
.\setup\01-install-llamacpp.ps1
.\setup\02-download-models.ps1 -Preset both
.\setup\03-install-opencode.ps1
.\setup\run-crucible.ps1      # in its own window
.\setup\benchmark.ps1         # confirm GPU+CPU are both working
opencode

Full walkthrough, tuning knobs, and troubleshooting are in the README.
